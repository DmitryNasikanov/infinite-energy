## Обоснование исправлений

**Холостые выстрелы** - тестирование системы на пониженной мощности:
- Масса: 10 кг (вместо 116 кг)
- Скорость: 1.5 км/с (вместо 5 км/с)
- Ускорение: ~15g (a = v²/2s = 1500²/(2×3000) = 375 м/с² ≈ 38g → округлим до 15g на половине трека)

Для получения 1.5 км/с при 15g (147 м/с²):
- Длина разгона: s = v²/(2a) = (1500)²/(2×147) = 7653 м = 7.6 км
- НО: можно использовать только часть трека (~1.5 км) или пониженное ускорение

Пересчёт с 15g на 1.5 км разгона:
- v = √(2as) = √(2×147×1500) = √441000 = 664 м/с

Альтернатива: оставить 3 км трек, но ускорение ~38g для достижения 1.5 км/с:
- a = v²/(2s) = (1500)²/(2×3000) = 375 м/с² = 38g

Выбрано: **1.5 км/с при пониженном ускорении ~15g на части трека** - логичный тест подсистем.

Энергия холостого выстрела:
- KE = ½mv² = ½ × 10 × (1500)² = 11.25 МДж = 0.01125 ГДж
- С 40% КПД: 0.01125 / 0.4 = 0.028 ГДж ~ 0.03 ГДж
