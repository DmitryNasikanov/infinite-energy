# Security Policy

This repository contains documentation and scientific content (not executable software), so traditional security vulnerabilities are unlikely. However, if you discover a security concern (e.g., exposed credentials, malicious links, or CI/CD misconfiguration), please report it responsibly.

## Reporting

- Use [GitHub Security Advisories](https://github.com/DmitryNasikanov/infinite-energy/security/advisories/new) for private disclosure
- Or open a regular [issue](https://github.com/DmitryNasikanov/infinite-energy/issues) if the concern is not sensitive
