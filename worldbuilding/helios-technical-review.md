# Часть I. Резюме

## Executive Summary

### Суть проекта

Проект ГЕЛИОС — программа создания **Роя Дайсона** на орбите Меркурия: системы из сотен миллионов автономных зеркал, перехватывающих 1% солнечной энергии и передающих её на Землю и другие планеты.

### Ключевые цифры

| Параметр | Значение |
|----------|----------|
| **Результат** | 2 000 ТВт электроэнергии на приёмниках Земли |
| **Сравнение** | 100× текущего мирового потребления (20 ТВт) |
| **Срок реализации** | 10 лет (6 подготовка + 4 на Меркурии) |
| **Бюджет** | $150–400 млрд (сопоставимо с Apollo) |
| **Инфраструктура** | 1 000 масс-драйверов, 1 000 заводов, 58 000 роботов |

### Почему реализуемо

1. **Физика работает.** Давление света для управления зеркалами (IKAROS, 2010), масс-драйверы (NASA, 1992), электролиз расплава (MRE) — известные технологии.

2. **Меркурий идеален.** Солнечный поток 6.7× выше земного. Нет атмосферы (нет сопротивления для масс-драйвера). Богат железом, кремнием, алюминием.

3. **Саморепликация масштабируема.** Роботы строят роботов. 99% массы производится на месте. С Земли — только «витамины» (электроника, редкие элементы): ~0.5% массы системы.

4. **Логистика решаема.** ~1 000 пусков за 4 года (при средней загрузке 20 т). SpaceX уже делает 100 пусков/год.

### Критические допущения

| Допущение | Риск | Митигация |
|-----------|------|-----------|
| Starship выходит на проектную мощность | Средний | Альтернативные носители (Long March 9, SLS) |
| Автономные роботы работают 24/7 | Высокий | Тестирование на Луне (Фаза -1) |
| Электролиз расплава (MRE) масштабируется | Средний | Земные прототипы (Фаза -2) |
| КПД передачи энергии 20%+ | Низкий | Проверено в лабораториях (power beaming) |

### Главный вывод

Проект не требует научных открытий — только инженерных решений на грани текущих возможностей. При бюджете уровня Apollo и международной кооперации реализуем в горизонте 10–15 лет.

# Часть II. Физика и инженерия

## 1. Физика проекта

### 1.1. Зеркала Роя

#### Проблема
Миллиарды зеркал в космосе. Нельзя использовать обычные двигатели (топливо кончится) или маховики (тяжёлые, ломаются).

#### Решение: Давление света + Электрохромика
Технология проверена японским солнечным парусником IKAROS (JAXA, 2010).

#### Конструкция зеркала

| Параметр | Значение |
|----------|----------|
| Размер | 100 × 100 м |
| Площадь | 10 000 м² |
| Материал | Алюминиевая фольга 4 мкм |
| Масса | ~150 кг |
| Мощность (отражённая) | 82 МВт |

**Форма:** Гигантский диск из фольги. Вращается вокруг оси (как фрисби) — центробежная сила натягивает фольгу, превращая её в жёсткий диск.

**Управление:** По краям — полоски из электрохромного материала (как в ЖК-часах):
- Включено → зеркальная поверхность (свет отражается, давит)
- Выключено → прозрачная поверхность (свет проходит насквозь)

**Маневрирование:** Чтобы повернуть зеркало, делаем один край прозрачным, другой — зеркальным. Возникает разница в давлении света → крутящий момент → зеркало поворачивается.

**Преимущества:**
- Нет движущихся частей
- Нет топлива
- Вечный ресурс (пока не деградирует электроника от радиации)

#### Термическая стойкость

Парадоксальный результат: зеркало холодное, не горячее.

```
Солнечный поток на Меркурии: 9 кВт/м²
Отражательность: 95%
Поглощение: 5% = 450 Вт/м²

Равновесная температура: −8°C
Температура плавления алюминия: 660°C
Запас: 668°C
```

Ключевой принцип: высокая отражательность фронтальной стороны (95%) + высокая излучательность тыльной (0.8) = эффективное охлаждение.

### 1.2. Энергетический Мост

#### Проблема
Расстояние Меркурий → Земля: 100–200 млн км. Простой луч рассеется.

#### Решение: Каскад из трёх прыжков

```
Меркурий → [Прыжок 1] → Космос → [Прыжок 2] → Орбита Земли → [Прыжок 3] → Поверхность
```

**Прыжок 1: Сбор (Меркурий → Космос)**
- Рой зеркал выстраивается в гигантскую фазированную решётку (~100 км)
- Жёсткий инфракрасный лазер
- Цель: Станция-Хаб на орбите Земли

**Прыжок 2: Транзит (Космос → Орбита Земли)**
- Приёмник на геостационарной орбите (36 000 км)
- Безопасность: если ошибка наведения — выключаем Хаб, луч уходит в пустоту

**Прыжок 3: Спуск (Орбита → Поверхность)**
- Многопучковая система (не один смертельный луч, а тысячи тонких)
- Адаптивная оптика для прохождения атмосферы
- Приёмники в пустынях (Сибирь, Гоби, Сахара)

#### КПД передачи

| Этап | КПД |
|------|-----|
| Отражение зеркал | 90% |
| Концентрация | 95% |
| Свет→лазер | 60% |
| Вакуум | 99% |
| Ретрансляция | 95% |
| Атмосфера | 95% |
| Приёмник (PV) | 50% |
| **Общий КПД** | **23%** |

| Целевая скорость | 5 км/с (вторая космическая + 16%) |

**Материалы (1 МД):**

### 2.2. Геология и ресурсы Меркурия

Меркурий — аномальная планета. Гигантское металлическое ядро, лишь слегка присыпанное каменной коркой.

#### Изобильные элементы

| Элемент | Содержание | Применение |
|---------|------------|------------|
| Железо (Fe) | ~70% ядра | Сталь, чугун — 90% массы роботов |
| Кремний (Si) | ~20% коры | Солнечные панели, стекло, керамика |
| Сера (S) | Аномально высокое | Изолятор, аккумуляторы (NaS) |
| Алюминий (Al) | Достаточно | Лёгкие корпуса, зеркала |
| Титан (Ti) | Есть в ильменитах | Особо прочные узлы |
| Натрий/Калий | Из полевых шпатов | Теплоноситель (жидкий металл) |

#### Дефицитные элементы («Витамины» с Земли)

| Элемент | Проблема | Решение |
|---------|----------|---------|
| Углерод (C) | Почти отсутствует | Нет пластиков, резины — везём |
| Водород (H) | Только лёд на полюсах | Добыча из кратеров |
| Медь (Cu) | Мало | Провода из алюминия/натрия |
| Вольфрам (W) | Критичен для резцов | Везём с Земли |
| Редкоземы | Критичны для электроники | Везём с Земли |
| Аргон/Азот | Отсутствуют | Везём с Земли |

### 2.3. Энергетический цех

| Локация | Поток |
|---------|-------|
| Орбита Земли | 1.3 кВт/м² |
| Меркурий | 9–14 кВт/м² |

#### Гелио-Башни (Power Towers)

Зеркала-концентраторы на «Пиках вечного света» (края кратера).

| Параметр | Значение |
|----------|----------|
| Материал | Полимерная плёнка (Каптон) + алюминиевое напыление |
| Вес | 50 г/м² |
| Общий вес комплекта | < 1 тонны |
| Фотоячейки | Многопереходные GaAs |

**Охлаждение:** Жидкий NaK циркулирует внутри мачты, сбрасывает тепло через радиатор в холодный космос.

**Сверхпроводящая ЛЭП:** Кабель по теневой стороне склона. В холоде сопротивление падает почти до нуля.

### 2.4. Производство солнечных панелей

Для масштабирования нужно местное производство кремниевых панелей (не GaAs).

| Параметр | GaAs | Кремний |
|----------|------|---------|
| КПД | 30-40% | 18-22% |
| Сырьё | Галлий + Мышьяк (дефицит) | **20% коры Меркурия** |
| Вердикт | Первый завод | **Масштабирование** |

#### Алюмотермическое восстановление кремния

Проблема: традиционный метод требует углерода, которого на Меркурии нет.

**Решение:** Алюмотермия.

```
3SiO₂ + 4Al → 3Si + 2Al₂O₃
```

На 1 кг кремния: 1.3 кг алюминия. Побочно: корунд (Al₂O₃) → керамика.

#### Энергопотребление 1 комплекса

| Потребитель | Мощность |
|-------------|----------|
| Масс-драйвер (средняя) | 39 МВт |
| Солнечные печи (плавка) | 10 МВт |
| Роботы + прочее | 6 МВт |
| **ИТОГО** | **~55 МВт** |

#### Площадь панелей на 1 комплекс

```
55 МВт / (9 кВт/м² × 18%) = 35 000 м² (квадрат 187×187 м)
```

### 2.5. Плавка и разделение

#### Солнечная печь (Solar Furnace)

Греем не купол, а тигель на улице. Вакуум = идеальная термоизоляция.

#### Электролиз расплава (MRE Cell)

В ванну с лавой опускаются электроды.

| Продукт | Судьба |
|---------|--------|
| Кислород | Сжижается → хранится как окислитель |
| Сплав (Fe/Si) | Оседает → рафинирование → МГД-ввод |
| Шлак (Al/Mg) | Всплывает коркой → скиммер → линия алюминия |

#### Разделение магния и алюминия

Mg кипит при 1091°C → возгоняется → конденсируется отдельно.

#### Дистилляция серы

| Параметр | Значение |
|----------|----------|
| Источник | Сульфиды в реголите (троилит FeS) |
| Метод | Вакуумная возгонка при 450°C |
| Применение | NaS-аккумуляторы |

### 2.6. Транзит через стену купола

Как передать расплав (1500°C) через холодный кратер (−150°C) внутрь завода?

#### Электромагнитный ввод

Труба с жидким металлом — «сэндвич» (Керамика + Индукторы + ЭВТИ).

| Компонент | Функция |
|-----------|---------|
| МГД-насос | Магнитное поле толкает металл. Нет подвижных деталей |
| Пробка | Из самого металла — герметичность |

#### U-образный гидрозатвор (Сифон)

Труба делает глубокий изгиб вниз. В нижней части U-трубы — «пробка» из тяжёлого железа, блокирующая выход газа из купола.

## 7. Проблемы масштабирования

### 7.1. Проблема электроники

```
Масса робота: 500 кг
«Витамины» (электроника): 3% = 15 кг
Роботов для Роя: 200 млн

ИТОГО: 200 млн × 15 кг = 3 000 000 тонн электроники
```

**Мировое производство пластин:** ~100-200 тыс. тонн/год.

**Вывод:** Требуется в 15-30 раз больше электроники, чем производит весь мир за год!

---

### 7.2. Проблема логистики

```
Груз: 3 000 000 тонн
Грузоподъёмность Starship: 100 тонн

Пусков нужно: 30 000
За 10 лет: 3 000 пусков/год
SpaceX сейчас: ~100 пусков/год

Нужно: 30× текущих возможностей
```

---

### 7.3. Проблема материалов

| Материал | Проблема | Масштаб |
|----------|----------|---------|
| Медь | Не обнаружена на Меркурии | Миллионы тонн для обмоток |
| Полимеры (Каптон) | Нет органики | 20 млн тонн для зеркал |
| Вольфрам | Критичен для резцов | 60 млн тонн для замены зубьев |

---

### 7.4. Решение: Две фазы

#### Фаза 1: «Плацдарм» (3-5 лет)

Ограниченный масштаб:
- 100-200 заводов
- ~200 000 роботов
- Электроники: 3 000 тонн (реалистично)
- Пусков: 30-50 (реалистично)

#### Фаза 2: «Муравейник» (15+ лет)

Переключение на местное производство:

1. **Примитивные чипы (180-300 нм)** — кремний на Меркурии есть
2. **«Королева»** — один суперкомпьютер управляет «тупыми» роботами
3. **Алюминиевая фольга** вместо Каптона
4. **Алюминиевые обмотки** вместо меди

**Импорт во Фазе 2:** ~0.01% от Фазы 1.

---

### 7.5. Реальная структура витаминов (Плацдарм)

Для 1 000 заводов, 1 000 МД, 58 000 роботов:

| Компонент | Масса с Земли | Доля |
|-----------|---------------|------|
| Масс-драйверы (электроника) | 7 000 т | 35% |
| Заводы (электроника + редкие металлы) | 5 000 т | 25% |
| Расходники (аноды, фильеры, газы) | 2 000 т | 10% |
| Зеркала (чипы управления) | 1 500 т | 7.5% |
| Роботы (бортовые системы) | 60 т | <1% |
| **Итого** | **~20 000 т** | |

**~45% грузов — электроника.** Единственный способ снизить зависимость — местное производство чипов.

## Раздел 9. Терраформирование Марса

---

### 9.1. Проблема Марса

| Параметр | Марс | Земля |
|----------|------|-------|
| Магнитное поле | **Нет** (потеряно 4 млрд лет назад) | 25-65 мкТл |
| Атмосфера | 0.6% земной | 100% |
| Температура | −60°C средняя | +15°C |
| Вода | Заморожена | Жидкая |

**Без магнитного поля:**
- Солнечный ветер сдувает атмосферу (~100 г/с)
- Поверхность облучается радиацией
- Любая созданная атмосфера улетит

---

### 9.2. Логика терраформирования

```
Шаг 1: Магнитный щит
        ↓
Атмосфера перестаёт улетать
        ↓
Шаг 2: Нагрев зеркалами
        ↓
CO₂ в полярных шапках испаряется
        ↓
Шаг 3: Парниковый эффект
        ↓
Температура растёт → ещё больше CO₂ испаряется
        ↓
Шаг 4: Лёд тает
        ↓
Жидкая вода на поверхности
        ↓
Шаг 5: Биосфера
        ↓
Растения производят кислород
```

---

### 9.3. Магнитный щит

#### Почему не минималистичный диполь в L1

Концепция NASA (2017) — диполь в точке L1:
- Мощность: ~150 МВт
- Проблема: Это «зонтик», а не полноценная магнитосфера
- Не защищает от космических лучей
- Не создаёт радиационные пояса

#### Полноценная магнитосфера

**Вариант: Орбитальное кольцо сверхпроводящих магнитов**

| Параметр | Значение |
|----------|----------|
| Орбита | Экваториальная, 400-1000 км |
| Количество спутников | 100-1000 |
| Магнитный момент (суммарный) | ~10²⁰ А·м² |
| Поле на поверхности | ~10-50 мкТл |

```
Сравнение:
Земля: 8 × 10²² А·м² → 25-65 мкТл на поверхности
Наш щит: 10²⁰ А·м² → 10-50 мкТл на поверхности
= достаточно для отклонения солнечного ветра
```

**Энергия щита: ~100 ГВт** (постоянно на криогенное охлаждение)

---

### 9.4. Нагрев планеты

#### Цель: Испарить полярные шапки

```
Масса CO₂ в шапках: 4 × 10¹⁵ кг
Теплота сублимации CO₂: 571 кДж/кг
Энергия на испарение: 2.3 × 10²¹ Дж

При мощности 500 ТВт:
Время = 2.3 × 10²¹ / 5 × 10¹⁴ ≈ 53 дня
+ нагрев льда с −120°C до −78°C → ~70 дней
```

**Вывод:** При 500 ТВт направленного тепла — полярные шапки испарятся за ~2-3 месяца.

#### Как направить тепло

Зеркала Роя перенаправляют солнечный свет на полюса Марса:
- 500 ТВт = ~6 млн зеркал (100×100 м каждое)
- Это ~2.5% от Роя к году 3

---

### 9.5. Парниковый эффект

#### После испарения CO₂

```
Текущее давление Марса: 0.6 кПа (0.6% земного)
+ 4 трлн тонн CO₂: +10-20 кПа
Новое давление: ~10-20 кПа (10-20% земного)

Парниковый эффект: +30-50°C
Новая средняя температура: −60 + 40 = −20°C
```

**Ещё холодно, но лёд на экваторе начнёт таять летом.**

### 9.6. Энергобюджет терраформирования

| Задача | Мощность | % Роя | Время |
|--------|----------|-------|-------|
| Магнитный щит | 100 ГВт | 0.001% | Постоянно |
| Испарение шапок | 500 ТВт | 5% | 2-3 месяца |
| Поддержание тепла | 500-1000 ТВт | 5-10% | 100+ лет |
| Электролиз воды (O₂) | 100 ТВт | 1% | Постоянно |
| Промышленность | 50 ТВт | 0.5% | Постоянно |

**Суммарно на Марс: ~15-17% мощности Роя**

Остальные 85% доступны для энергоснабжения Земли, других проектов, межпланетной логистики.
