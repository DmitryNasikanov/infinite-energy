Риски и ограничения проекта

TL;DR

  • Решённых рисков: 5
    • Синхронизация фазированной решётки → локальные кластеры
    • Электрохромика → достаточна для задачи
    • Медь на Меркурии → криоалюминий в тени
    • NaS-батареи → кабели + станции
    • Орбитальная механика → давление света + эфемериды
  • Открытых критических: 1 (иридий для анодов → Cr-Fe-Co как основной путь)
  • Высоких рисков: 5 (влияют на параметры, не блокируют проект)
    • КПД энергомоста → избыточность 100×
    • Деградация зеркал → стратегия “добавляй, не заменяй”
    • МГД-насос → снят (ошибка в оценке)
    • Тепловое расширение → углепластик (импорт)
    • Ресурсы Меркурия → MESSENGER подтверждает
    • Время сборки → задержка ~2% графика
  • Ресурсных ограничений: 3 (лимиты мировой промышленности)
  • Неопределённостей: 6 (требуют исследований)

Решённые критические риски

Эти проблемы изначально блокировали проект, но теперь имеют решения.

1. Синхронизация фазированной решётки

Проблема: Для коллимированного лазерного луча на 100 млн км нужна синхронизация миллиарда зеркал с точностью в наносекунды. Задержка сигнала туда-обратно: ~670 секунд — невозможно замкнуть feedback loop.

Источник: Проект за 5 минут, Передача энергии

Решение: Локальные кластеры

Вместо глобальной когерентности используем группы из 1000–10 000 зеркал, синхронизированные внутри кластера:

Параметр Глобальная решётка Локальный кластер
База D 10⁶ км 100 км
Задержка связи 670 сек 0.3 мс
Синхронизация Невозможна ✅ Реализуема
Расширение луча ~1 м ~1 км

Подробности: Координация миллиарда зеркал

Статус: ✅ Решено (локальные кластеры)

2. Электрохромика для управления ориентацией

Проблема: IKAROS использовал электрохромику для грубого управления (~1°). Для фазированной решётки нужна наносекундная синхронизация — физически невозможно с электрохромикой (время отклика ~100 мс).

Источник: Передача энергии

Решение: Электрохромика достаточна

При переходе на локальные кластеры (см. #1) требования к точности снижаются:

  • Нужна ориентация ~0.01° (не наносекундная синхронизация)
  • Электрохромика IKAROS обеспечивала ~1° — с запасом
  • Давление света: ~60 мкПа на орбите Меркурия → достаточно для точного позиционирования

Подробности: Система наведения

Статус: ✅ Решено (электрохромика достаточна для кластерной архитектуры)

3. Медь на Меркурии не найдена

Проблема: Для масс-драйверов нужны обмотки с высокой проводимостью. Меди на Меркурии не обнаружено (MESSENGER).

Источник: Проект за 5 минут, Масс-драйвер

Расчёт влияния: - Алюминий: 60% проводимости меди → КПД падает с 40% до ~30% - Натрий в стальных трубках: ~40% проводимости меди, но реактивен

Решение: Криоалюминий (естественное охлаждение)

Масс-драйверы размещены стационарно с оптимизацией по максимальному времени в тени. Большую часть меркурианского дня (176 земных суток) конструкция находится в тени при -180°C — алюминий естественно криогенный без активной системы охлаждения:

Конфигурация КПД Условия
Алюминий при +20°C (солнечный период) 25-30% Простой
Алюминий при -180°C (теневой период) 35-40% Естественное охлаждение
Импорт меди 40% Дорого

При -180°C проводимость алюминия возрастает до 360% от номинала — сопоставимо с медью при комнатной температуре. Солнечные панели размещены рядом, на освещённой стороне.

Подробности: Криоалюминий

Статус: ✅ Решено (стационарное размещение в тени)

4. Ресурс NaS-батарей в ночной период

Проблема: Ночь Меркурия = 88 земных суток. NaS-батареи требуют подогрева (300°C), потребляя ~5-10 кВт постоянно.

Источник: Бестиарий роботов

Расчёт:

Энергия на подогрев: 10 кВт × 88 × 24 ч = 21 МВт·ч
Ёмкость батареи робота: ~10-15 кВт·ч
Результат: Батарея разряжается за 1-2 часа

Решение: СИСТЕМА ЭНЕРГОСНАБЖЕНИЯ ДЛЯ ВСЕХ GEN-2 РОБОТОВ

Робот Решение Принцип
Крот-М Кабель 10 км + автономия Радиальная добыча от завода
Краб-М Зарядные станции Маршрутная логистика, зарядка при погрузке
Кентавр-М Кабель 50 м (внутри) + станции (снаружи) Две зоны работы, вахтовый режим

Физика терминатора: - Скорость терминатора: ~3 м/с (медленнее пешехода) - Температура в зоне терминатора: 0–100°C (идеально для роботов)

Инфраструктура на 1 комплекс: ~35–37 точек (кабельные + зарядные станции)

Подробности: - Операционная модель Крот-М - Операционная модель Краб-М - Операционная модель Кентавр-М - Инфраструктура энергоснабжения

Статус: ✓ Решено для всех Gen-2 роботов

Открытые критические риски

Требуют дополнительной проработки.

5. Иридий для анодов MRE

Проблема: Аноды MRE расходуются и требуют замены каждые 6 месяцев.

Статус технологии MRE: TRL 5/6 (NASA KSC 2024, Blue Origin Blue Alchemist, Lunar Resources — 25 кг реголита переработано). Сама технология электролиза подтверждена, проблема — только в ресурсе анодов.

Источник: Завод “Точка Ноль”, Технологическая зрелость

Расчёт: | Параметр | Значение | |———-|———-| | Масса анода | ~20 кг иридия | | Замен за 10 лет | 20 | | На 1000 заводов | ~400 тонн | | Мировое производство | ~7 т/год | | Дефицит | ~6× мирового производства |

Альтернативы (ранжированы по реалистичности):

Альтернатива TRL Ресурс анода Импорт Рекомендация
Cr-Fe-Co аноды ~4 3 мес 0 (Cr, Fe на Меркурии) Основной путь
Углеродные расходуемые ~3 Одноразовые C с полюсов (1-3% в LRM) Резерв
H₂ восстановление ~2 Не нужны H₂O с полюсов Долгосрочно

Основной путь — Cr-Fe-Co: - Ресурс ниже (замена каждые 3 мес вместо 6), но материалы доступны на Меркурии - Масса анода ~20 кг → 80 кг/год/завод × 1000 заводов = 80 т/год (местное производство) - Хром: содержание в реголите достаточное для масштаба

Статус: ⚠️ Серьёзная проблема — Cr-Fe-Co наиболее вероятное решение, требует валидации TRL

Высокие технические риски

Влияют на параметры проекта, но не блокируют реализацию.

6. КПД энергомоста завышен

В расчётах: 22-23% Реальная оценка: 10-15%

Этап Расчётный Реальный
Свет → лазер 60% 40-50%
Приёмник (PV) 50% 35-45%
Атмосфера (с облачностью) 85% 30-60%

Последствие: Реальная мощность на Земле в 1.5-2× ниже расчётной

Компенсация: Избыточность 100×

Проект изначально рассчитан с запасом:

Параметр Значение
Отражённая мощность 10 ПВт
При КПД 18% 1.8 ПВт = 1800 ТВт
При КПД 10% 1 ПВт = 1000 ТВт
Мировое потребление 20 ТВт
Запас 50-100×

Даже при пессимистичном КПД 10% → мощность = 50× мирового потребления.

Подробности: Итоговый КПД

Статус: ⚠️ Учтённое ограничение (компенсируется избыточностью)

Источник: Проект за 5 минут, Передача энергии

7. Деградация зеркал занижена

Параметр В расчётах Реальная оценка
Срок службы 10-15 лет 5-8 лет
Потери КПД/год 2-5% 5-10%

Причины: - Солнечный ветер на орбите Меркурия в 10× сильнее - Микрометеориты (JWST показал проблему) - Электростатика

Решение: Стратегия “добавляй, не заменяй”

Место в рое не ограничено. Старые зеркала продолжают работать на сниженном КПД, новые добавляются:

  • При полном отказе → автоматическая утилизация (падение на Солнце через солнечный парус)
  • Производство 219 млн/год компенсирует ускоренную деградацию

Пересчёт при сроке 7 лет (пессимистичный):

Год Произведено Выбыло Всего активных
1–6 548 млн 0 548 млн
7 219 млн 11 млн 756 млн
8 219 млн 33 млн 942 млн
9 219 млн 88 млн 1073 млн
10 219 млн 197 млн 1095 млн

Пиковый дефицит vs. план: ~15%. Целевая мощность 10 ПВт достигается к году 5 (без изменений), дальнейшее накопление замедляется.

Источник: Передача энергии, Зеркала

Статус: ⚠️ Влияет на темп накопления после года 7, не блокирует проект

8. Мощность МГД-насоса занижена → Снят

Исходная формулировка: “20 кВт в документе → 500 кВт реально, занижение 25×”

Причина снятия: Путаница компонентов. По данным секции плавки:

Компонент Мощность Что это
Магнитный сепаратор 20 кВт Исходные “20 кВт”
МГД-каналы 100 кВт (2×50) Собственно насос
Вся секция плавки 225 кВт МГД + индукция + NaK + автоматика
Электролиз MRE 500 кВт Исходные “500 кВт” (другой узел)

При бюджете завода ~50 МВт секция плавки = 0.45% потребления. Не является риском.

Статус: ❌ Снят (ошибка в исходной оценке)

9. Тепловое расширение зеркал

Параметр Значение
Температура освещённой стороны +38°C
Температура в тени −50°C
ΔT при вращении до 88 K
Коэффициент Al 23×10⁻⁶ K⁻¹
Смещение края 100 м зеркала ~20 см

Расчёт: 100 м × 23×10⁻⁶ K⁻¹ × 88 K = 0.20 м

Последствие: Тепловые деформации нарушают плоскостность и точность наведения. Для фазированной решётки критично.

Решение: Композитный каркас с низким КТР

Материал каркаса CTE (K⁻¹) Смещение на 100 м Импорт
Алюминий (текущий) 23×10⁻⁶ 20 см 0
Углепластик (CFRP) ~1×10⁻⁶ 0.9 см ~0.5 т/завод
Инвар (Fe-36Ni) ~1.2×10⁻⁶ 1.1 см Ni с полюсов?

Углепластик уменьшает деформацию в 23× → остаточное смещение ~1 см на 100 м, допустимо для кластерной архитектуры. Входит в категорию “витаминов” (импорт с Земли).

Подробности: Расчёт температуры зеркала

Источник: Проект за 5 минут, Энергия

Статус: ⚠️ Решаемо импортом углепластика (~0.5 т/завод)

10. Орбитальная механика Роя

Исходные опасения:

  • Нет модели возмущений от Венеры, Юпитера, Земли
  • Нет расчёта энергии на коррекцию орбит миллиарда объектов
  • Микрометеориты смещают центр масс зеркал

Решение: Давление света + эфемериды

Каждое зеркало — солнечный парус с автономной коррекцией:

Механизм Принцип
Давление света ~60 мкПа на орбите Меркурия → микрокоррекции без топлива
Электрохромика Разница отражательности → крутящий момент
Предсказуемые орбиты Эфемериды рассчитываются заранее
Кластерная структура Возмущения воздействуют на весь кластер равномерно

Зеркала на гелиоцентрических орбитах стабилизированы гравитацией Солнца. Возмущения от планет — медленные и предсказуемые.

Подробности: Удержание на орбите

Статус: ✅ Решено (давление света + эфемериды)

Источник: Проект за 5 минут

11. Ресурсы Меркурия — валидация по данным MESSENGER

Источники: MESSENGER XRS/GRS (2011-2015), Science, JGR Planets

Элемент В проекте MESSENGER Статус
Железо (Fe) ~5% 1.5–2% ⚠️ В 2.5× меньше
Кремний (Si) ~20% ~24.6% ✓ Достаточно
Сера (S) Высокое 2.5–4% ✓ Идеально (10× Земли)
Натрий (Na) Достаточно 2.6–4.9% ✓ Достаточно
Алюминий (Al) Достаточно Ниже Луны ⚠️ Умеренно
Магний (Mg) Достаточно Доминирует ✓ Бонус
Медь (Cu) Нет Нет ✓ Учтено
Углерод (C) Почти нет 1-3% в LRM ✓ Переоценён!

Железо в деталях:

Локация Содержание
Ядро ~70% (недоступно, глубина 400+ км)
Кора/поверхность 1.5–2%
Сравнение: Луна ~10%

Последствие: Нужно перерабатывать в 2.5–3× больше реголита для получения железа. НО: зеркала делаются из алюминия, железо нужно только для роботов и заводов (~1 млн тонн на весь проект).

Главное преимущество Меркурия — энергия (7×), не минералы.

Источник: NASA MESSENGER, исправлено в Проекте

12. Время сборки завода (7 дней) — оптимистично

В документе: 7 дней “блиц-монтаж” Реальная оценка: 30-60 дней

Причины: - Предсборка на Земле минимизирует, но не устраняет работу на месте - Непредвиденные проблемы при первой посадке - Калибровка и тестирование

Анализ последствий:

Сценарий Сборка Первый МД Сдвиг графика
Оптимистичный 7 дней Месяц 6 0
Реалистичный 30 дней Месяц 7 +1 мес
Пессимистичный 60 дней Месяц 8 +2 мес

Почему не критично: - Задержка одноразовая — только первый завод собирается Gen-1 роботами - Последующие заводы собираются Gen-2 роботами (быстрее, опытнее) - Экспедиция 2 стартует через 1 месяц, прибывает через 4-5 — задержка не влияет на неё - Сдвиг 1-2 мес при 10-летнем проекте = ~2% графика

Источник: Завод, Дорожная карта

Статус: ⚠️ Учтённое ограничение (не блокирует проект)

Пусковые мощности

Параметр Значение

Почему 33% выход (а не 8% Al)?

Используется MRE (Molten Regolith Electrolysis) — электролиз ВСЕГО расплавленного реголита:

Элемент В реголите (MESSENGER) После электролиза
Al ~8% ✓ Алюминий
Fe ~2% ✓ Железо
Si ~25% ✓ Кремний
Mg ~10% ✓ Магний
Итого полезных ~45%
С КПД 70% — ~32-33%
